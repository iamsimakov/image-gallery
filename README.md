# Image Gallery Rails Application

[![Greenkeeper badge](https://badges.greenkeeper.io/cmckni3/image-gallery.svg)](https://greenkeeper.io/)

## Requirements

* ImageMagick
* Ruby 2.4.1
* Rails 5.2.0
* Mysql server: user - root, empty password

## Setup

* Clone it

* Bundle install

```bash
bundle install
```
* Config your connect to db

```bash
config/database.yml
```

* Run it

```bash
rails s
```
